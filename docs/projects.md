# Projects

## Vectras — Personal Portfolio Site

A modern, bilingual personal website built with Docusaurus, featuring interactive React components for visualizing skills and experience.

- **Tech Stack**: Docusaurus 3, React 19, TypeScript, React Flow, Recharts, CSS Modules
- **Features**: i18n (English/Chinese), glassmorphism UI, interactive skill map, responsive design
- **Link**: [GitHub](https://github.com/WZhijun154/vectras)

## Autonomous Driving Planning Framework

An internal planning and control framework for L2+ autonomous driving, built at Horizon Robotics.

- **Tech Stack**: C++17, ROS 2, Eigen, OSQP
- **Highlights**: Real-time trajectory optimization with 50ms planning cycles, integrated safety constraints, scenario-based testing framework
- **Role**: Core developer for the motion planning module

## VLM Distributed Training Pipeline

A scalable training infrastructure for Vision-Language Models at SenseTime.

- **Tech Stack**: Python, PyTorch, DeepSpeed, Megatron-LM, SLURM
- **Highlights**: Multi-node distributed training across 100+ GPUs, mixed-precision training with gradient checkpointing, automated data pipeline for multimodal datasets
- **Role**: Lead engineer for training infrastructure

## ML-Enhanced Control System (Research)

Master's thesis project at Osaka University — applying machine learning to improve traditional control system performance.

- **Tech Stack**: Python, PyTorch, MATLAB/Simulink
- **Highlights**: Proposed a hybrid approach combining model-based control with learned residual dynamics, achieving improved tracking performance
- **Link**: Publication available upon request
