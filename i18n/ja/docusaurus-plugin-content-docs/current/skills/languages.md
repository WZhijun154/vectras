# 言語能力

中国語、日本語、英語の3言語を使いこなします。

## 中国語（普通話）

**母語** — 仕事でもプライベートでも日常的に使用。

## 日本語

**流暢（JLPT N1）** — 大阪大学での2年間の大学院生活を通じて習得。技術的な議論、学術的な執筆、日常会話に対応可能。

## 英語

**ビジネスレベル** — 技術論文の読解、ドキュメントの作成、国際チームとのコラボレーションに広く使用。ビジネス環境で問題なくコミュニケーション可能。
