# プロジェクト

## Vectras — 個人ポートフォリオサイト

Docusaurusで構築されたモダンなバイリンガル個人サイト。スキルと経験を可視化するインタラクティブなReactコンポーネントを搭載。

- **技術スタック**: Docusaurus 3、React 19、TypeScript、React Flow、Recharts、CSS Modules
- **特徴**: 多言語対応（英語/中国語/日本語）、グラスモーフィズムUI、インタラクティブスキルマップ、レスポンシブデザイン
- **リンク**: [GitHub](https://github.com/WZhijun154/vectras)

## 自動運転計画フレームワーク

Horizon Roboticsで開発されたL2+自動運転の計画・制御フレームワーク。

- **技術スタック**: C++17、ROS 2、Eigen、OSQP
- **ハイライト**: 50msの計画サイクルでのリアルタイム軌道最適化、統合安全制約、シナリオベースのテストフレームワーク
- **役割**: モーションプランニングモジュールのコア開発者

## VLM分散学習パイプライン

SenseTimeでのスケーラブルなVision-Language Model学習インフラ。

- **技術スタック**: Python、PyTorch、DeepSpeed、Megatron-LM、SLURM
- **ハイライト**: 100+ GPUでのマルチノード分散学習、混合精度学習と勾配チェックポイント、マルチモーダルデータセットの自動データパイプライン
- **役割**: 学習インフラのリードエンジニア

## ML強化制御システム（研究）

大阪大学の修士論文プロジェクト — 機械学習を応用した従来の制御システム性能の改善。

- **技術スタック**: Python、PyTorch、MATLAB/Simulink
- **ハイライト**: モデルベース制御と学習された残差ダイナミクスを組み合わせたハイブリッドアプローチを提案し、追従性能の向上を実現
- **リンク**: 論文はご要望に応じて提供可能
