# 项目

## Vectras — 个人作品集网站

一个现代化的双语个人网站，使用 Docusaurus 构建，包含交互式 React 组件来展示技能和经验。

- **技术栈**: Docusaurus 3、React 19、TypeScript、React Flow、Recharts、CSS Modules
- **特性**: 国际化（英文/中文）、毛玻璃 UI、交互式技能图、响应式设计
- **链接**: [GitHub](https://github.com/WZhijun154/vectras)

## 自动驾驶规划框架

在地平线机器人开发的 L2+ 级自动驾驶内部规划和控制框架。

- **技术栈**: C++17、ROS 2、Eigen、OSQP
- **亮点**: 50ms 规划周期的实时轨迹优化、集成安全约束、基于场景的测试框架
- **角色**: 运动规划模块核心开发者

## VLM 分布式训练管线

商汤科技的可扩展视觉语言模型训练基础设施。

- **技术栈**: Python、PyTorch、DeepSpeed、Megatron-LM、SLURM
- **亮点**: 100+ GPU 的多节点分布式训练、混合精度训练与梯度检查点、多模态数据集的自动化数据管线
- **角色**: 训练基础设施负责人

## 基于 ML 的增强控制系统（研究）

大阪大学硕士论文项目 — 应用机器学习改善传统控制系统性能。

- **技术栈**: Python、PyTorch、MATLAB/Simulink
- **亮点**: 提出了结合基于模型的控制与学习残差动力学的混合方法，实现了更好的跟踪性能
- **链接**: 论文可按需提供
